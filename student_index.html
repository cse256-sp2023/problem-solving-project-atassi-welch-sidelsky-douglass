In each scenario, you will see a simplified file system management interface.
Pretend that you are an administrator for this file system, and you are tasked with maintaining correct file permissions.
Complete the task given in each scenario.

Easy

1. Remove some permissions from employee (V1) (remove_direct_permission)
2. Add permissions for a new employee (add_new_user)
3. Add permissions for a manager (add_full_permissions)

Medium

4. Remove some permissions from employee (V2) (remove_inherited_permission)
5. Add permissions for an intern (intern_permissions)
6. Remove ALL permissions from employee (remove_user_with_inheritance)

Hard

7. Remove some permissions from employee (V3) (restrict_group_member)
8. Fix permissions for a teaching assistant (let_ta_modify)
9. Fix permissions for all users (lost_inheritance)
